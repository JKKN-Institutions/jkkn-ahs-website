# Library Images Directory

This directory contains images for the Library page at `/facilities/library`.

## Required Images

Place the following library images in this directory:

1. **library-main.jpg** (or .png)
   - Main library entrance or overview shot
   - Recommended size: 1920x1080 px or similar 16:9 ratio
   - Used as hero image on the page

2. **library-reading-area.jpg** (or .png)
   - Reading areas and study spaces
   - Recommended size: 1600x900 px or similar 16:9 ratio
   - Shows the study environment

3. **library-collection.jpg** (or .png)
   - Digital resources and book collections
   - Recommended size: 1600x900 px or similar 16:9 ratio
   - Showcases the library resources

## Image Specifications

- **Format**: JPG, PNG, or WebP
- **Aspect Ratio**: 16:9 (landscape)
- **File Size**: Optimize to under 500KB for better performance
- **Quality**: High resolution for clarity on all devices

## After Adding Images

Once you've placed the images in this directory, update the Library page component:
- File: `src/app/facilities/library/page.tsx`
- Follow the instructions in the LIBRARY PAGE IMAGE GUIDE comment at the top of the file
- Replace the placeholder divs with the actual Image components

## Example Image Sources

You can obtain library images from:
- Professional photography of your library
- Stock photo websites (ensure proper licensing)
- University/college media resources
